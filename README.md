## classLoader.py
### Import 
Contains classes to import data from national grid 'Electricity System Operator (ESO) of Great Britain (GB) and from 'Bundesnetzagentur' in Germany.
- class ESOImport 
    - method combine_eso
- class SMARDImport
    - method combine_smard

#### Tidy
Contains class for an initial data structuring by printing information of the raw data and standardizing the time stamps for further processing
- class Tidy
    - method df_info
    - method standardize_time
